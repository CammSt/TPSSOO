PRUEBA 1 Prueba Planificación Corto Plazo

Iniciar los módulos.
    Parámetros del Kernel   
        archivo_pseudocodigo: PLANI_CORTO_PLAZO
        tamanio_proceso: 0
Definir el algoritmo de corto plazo a FIFO con dos CPU. Cuando se están ejecutando los procesos infinitos levantar un segundo IO Disco. Luego de verificar que se estén usando los dos IO, matar ambos.
Cambiar el algoritmo de corto plazo a SJF con un solo CPU y volver a ejecutar. Una vez que queden los procesos infinitos matar la IO Disco.
Cambiar el algoritmo de corto plazo a SRT con un solo CPU y volver a ejecutar. Una vez que queden los procesos infinitos matar la IO Disco.

./bin/memoria ./MEMORIA_CP.config
./bin/kernel ./KERNEL_CP_FIFO.config PLANI_CORTO_PLAZO 0
./bin/cpu ./CPU_CP.config CPU1
./bin/cpu ./CPU_CP.config CPU2
./bin/io ./io.config DISCO
./bin/io ./io.config DISCO

./bin/memoria ./MEMORIA_CP.config
./bin/kernel ./KERNEL_CP_SJF.config PLANI_CORTO_PLAZO 0
./bin/cpu ./CPU_CP.config CPU1
./bin/io ./io.config DISCO

./bin/memoria ./MEMORIA_CP.config
./bin/kernel ./KERNEL_CP_SRT.config PLANI_CORTO_PLAZO 0
./bin/cpu ./CPU_CP.config CPU1
./bin/io ./io.config DISCO
